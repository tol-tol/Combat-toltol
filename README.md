# Minecraft Anti-Logout During Combat System

Features

- When player-versus-player combat begins, players automatically enter Combat Mode.

- While in Combat Mode, the remaining combat time is displayed in the action bar.

- Combat Mode automatically ends when a winner is decided or when the combat timer expires.

- If a player logs out of the server or uses teleportation during Combat Mode, they will die instantly.

- During Combat Mode, players cannot use commands or open chests.

OP Command
```
/combatmode set <time_in_seconds>


This command allows operators to set the duration of Combat Mode (in seconds).
```


# 마인크래프트 전투 중 탈주 방지 시스템

# 기능
- 플레이어 대 플레이어 전투가 시작되면 플레이어는 자동으로 전투 모드에 들어갑니다.
 
- 전투 모드에서는 남은 전투 시간이 액션 바에 표시됩니다.

- 전투 모드는 승자가 결정되거나 전투 타이머가 만료되면 자동으로 종료됩니다.

- 전투 모드 중에 플레이어가 서버에서 로그아웃하거나 텔레포트를 사용하면 즉시 사망합니다.

- 전투 모드에서는 플레이어가 명령을 사용하거나 상자를 열 수 없습니다.




### OP command

```
/전투모드 설정 [시간초]

이 명령어를 사용하면 운영자는 전투 모드의 지속 시간을 초 단위로 설정할 수 있습니다.
```
